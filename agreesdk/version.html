    # 更新日志

    # AgreeSDK 2018-06-08 发布日志(v1.0.7)

    • Device 模块增加 isLockTask 方法，可以判断设备是否被锁定。
    • 原本创建地图的方法传入Dom节点的id，改为可传入Dom节点的id也可为Dom节点。
    • Amate相关包在thirdpart/package下，若是Amate jar包更新，替换掉T5包和so库即可。(前提是Amate设备文档接口保持不变)
    • 视频使用ffmpeg压缩处理。
    • file模块新增创建目录方法和获取工作空间、缓存空间、sd卡根目录路径等属性。

    # AgreeSDK 2018-05-28 发布日志(v1.0.6)

    • 修复地图 hide 方法不能正常隐藏的bug，增加 destroy 方法。
    • 添加AMate背夹支持，详情请看docs/AgreeSDK.extdevice.html文档。
    • AgreeSDK.window添加修改状态栏字体颜色的功能(Android6.0及以上支持)，详情看文档。
    • 调试基座新增动态配置沉浸式状态栏、开机自启动、控制台的配置功能。
    • 修复Device模块中禁用摄像头、截图的方法有歧义错误，使用禁用摄像头和禁用截图的功能时请看文档。
    • 修复打开外部链接时，返回功能的异常。

    # AgreeSDK 2018-05-14 发布日志(v1.0.5)

    • 重要：调整 AgreeSDK JS 代码组织方式，保持多个渠道包管理一致性，支持后续采用 npm 包组织代码，组件 API 调用方式不受影响
    • 新增指纹识别功能，文档参考：http://192.9.200.185:50003/agreesdk-doc/AgreeSDK.fingerprint.html
    • 新增开机自启动功能，文档参考：http://192.9.200.185:50003/agreesdk-doc/index.html
    • 新增推送资源热更新功能，文档参考：http://192.9.200.185:50003/agreesdk-doc/index.html
    • 新增 workspace@www 协议文件路径，例如：workspace@www:///static/browser.html
    • 修复 input 输入框无图标、不能输入的bug
    • 调试基座新增 IP 快捷调试的功能，避免多次扫码
    • 原生层异常拦截输出到控制台

    # AgreeSDK 2018-04-27 发布日志(v1.0.4)

    • 修复 manifest.json launchPath 路径引用不正常的 bug
    • 确认 websocket 使用发现异常的问题原因，建议通过 Vue.use 引用 AgreeSDK.js
    • 新增离线地图包下载管理和地图截图功能
    • 新增 Video 模块视频录制功能

    # AgreeSDK 2018-04-24 发布日志(v1.0.3)

    • 支持沉浸式状态栏，详细配置见文档说明
    • 修复 config.xml 中配置多余插件，类 java.lang.ClassNotFoundException 的异常.
    • 修复错误页面关闭按钮不生效的 bug
    • File 模块增加 PDF 查看的方法
    • notification 模块增加设置和清除应用快捷方式上显示的提示数字 addBadgeNumber 和 clearBadgeNumber 方法
    • Storage 模块增加数据库操作 execSQL 和 rawQuery 方法
    • Map 模块增加 Marker 标记、POI 检索、路线规划 和 用系统第三方程序进行导航的相关方法。

    # AgreeSDK 2018-04-16 发布日志(v1.0.2)

    • 优化启动页关闭逻辑，增加 AgreeSDK.app.closeSplashScreen(); 方法在 JS 层关闭启动页；
    • 优化资源更新逻辑，配置 manifest.json abs 字段设置资源更新相关参数；
    • 增加 WebSocket 组件和基于 WebSocket 组件的 push 组件；
    • 增加对 file:// 协议下 WebRTC 的支持；

    # AgreeSDK 2018-04-04 发布日志(v1.0.1)

    • 提供侧滑 Debug 调试支持，在 release 工程下，从屏幕右侧边缘向左滑动，即可打开 Debug 调试。Debug 调试可显示 log 信息以及输入 js 调试等功能，可以通过 manifest.json debugable 启用与关闭；
    • 新增 speech 插件，提供了语音合成和语音识别两个功能；
    • 新增 storage 插件，提供了 SQLite 数据库的增删改查功能和本地键值对存储的功能；
    • 新增 network 插件，提供了网络的相关信息。

    # AgreeSDK 2018-03-23 发布日志(v1.0.0)

    • AgreeSDK.js 采用 ES6 编写，提供 JS 与 Native 交互的通信机制，发布者订阅者模式的事件通道机制以及插件的拓展机制；
    • 内置 Window 插件，提供多窗口的栈结构设计，原生下拉刷新支持，窗口通信机制，原生 TitleBar 支持，以及侧滑 Debug 调试支持；
    • 内置 application、audio、barcode 、device、file、image map、nativeUI、nativeObject、notification、screen 基本插件；
    • 提供 AnyOffice、imate、sdses 三方组件。
